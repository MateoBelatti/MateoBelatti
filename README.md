## 🚀 Proyectos

### 🕯️ Canela Artesanías — E-commerce de productos artesanales

> React · Vite · TypeScript · C# · ASP.NET Core · EF Core · SQL Server

Tienda online completa con catálogo de productos, carrito de compras y panel de administración protegido por roles.

**Backend:** arquitectura en capas (`Api` / `Services` / `Repository` / `Biblioteca`) con patrón Repository sobre EF Core. Autenticación con JWT + Google OAuth. Endpoints RESTful con manejo centralizado de errores. Desplegado en Railway.

**Frontend:** arquitectura por features con separación en `pages`, `components`, `hooks`, `services` y `context`. Carrito de compras implementado con Context API, rutas protegidas por rol y formularios con validación tipada. Desplegado en Vercel.

🌐 [Ver demo](https://canelaartesanias.vercel.app/) · ⚙️ [Backend](https://github.com/MateoBelatti/AppShowroom-Backend) · 🖥️ [Frontend](https://github.com/MateoBelatti/AppShowroom-Frontend)

### 🔍 Ingened — Gestión de informes de inspección técnica

> React · Vite · TypeScript · C# · ASP.NET Core · EF Core · SQL Server

Herramienta interna para empresas de inspección no destructiva. Permite crear, completar y exportar informes técnicos en PDF, con almacenamiento automático en Google Drive vía Service Account.

**Backend:** misma arquitectura en capas que Canela, con lógica de negocio encapsulada en la capa de servicios. Integración con Google Drive API usando credenciales de Service Account. Generación de PDFs server-side.

**Frontend:** formularios dinámicos por sección con patrones de manejo de estado compartidos, tipos centralizados y constantes reutilizables. Arquitectura orientada a mantenibilidad dado el alto volumen de campos por informe. Desplegado en Vercel.

🌐 [Ver demo](https://ingenedpdf.vercel.app/) · ⚙️ [Backend](https://github.com/MateoBelatti/Ingened-Backend) · 🖥️ [Frontend](https://github.com/MateoBelatti/Ingened-Frontend)
